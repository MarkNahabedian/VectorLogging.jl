VectorLogging.jl

VectorLOgging provides an in-memory logging destination.

VectorLogging.LogEntry — Type
LogEntry

LogEntry encapsulates the data for a single log message to VectorLogger.

source
VectorLogging.VectorLogger — Type
VectorLogger

VectorLogger is an in-memory log destination. The logger's log slot is a Vector of LogEntrys.

source
using Logging
using VectorLogging

logger = VectorLogger()

with_logger(logger) do
    @info "Message 1"
    @info "Message 2"
end

logger.log
2-element Vector{LogEntry}:
 LogEntry(Info, "Message 1", Main, :index, :Main_Main_d2d482e3, "index.md", 18, Base.Pairs{Symbol, Union{}, Tuple{}, NamedTuple{(), Tuple{}}}())
 LogEntry(Info, "Message 2", Main, :index, :Main_Main_5ac2d681, "index.md", 19, Base.Pairs{Symbol, Union{}, Tuple{}, NamedTuple{(), Tuple{}}}())
